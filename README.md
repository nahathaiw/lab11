# Lab11 - Physics Platform Game

A Cocos Creator platform-game exercise focused on physics, ray casting, timed input, moving platforms, and collision callbacks. The player automatically walks between walls and charges jump power by holding the mouse.

## What the project demonstrates

- Cocos 2D physics with rigid bodies, gravity, ray casts, and contact callbacks
- Hold-and-release jump charging with an energy bar
- Procedurally generated normal and conveyor platforms
- Moving-platform behavior and one-way platform collisions
- Camera tracking, looping backgrounds, score tracking, pause controls, sound, and game-over flow

## Technology

- Cocos Creator 2.4.8
- TypeScript

## Run locally

Open the repository in Cocos Creator 2.4.8, load `assets/Scenes/game.fire`, and run Preview.
